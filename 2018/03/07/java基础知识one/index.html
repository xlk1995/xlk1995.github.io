Java的起源

Java刚开始的名字叫做Oak，据说是它的创始人看窗外的时候正好有一颗橡树而起名的，它目标设定在家用电器等小型系统的编程语言，来解决诸如电视机、电话、闹钟、烤面包机等家用电器的控制和通讯问题。由于这些智能化家电的市场需求没有预期的高，Sun放弃了该项计划。就在Oak几近失败之时，随着互联网的发展，Sun看到了Oak在计算机网络上的广阔应用前景，于是改造了Oak，以“Java”的名称正式发布。

Java的特性

Java的第一个最出名的特性就是可移植性/跨平台，它可以在各种操作系统里面运行。java源文件编译成.class文件，通过jvm（Java Virtual Machine）在不同操作系统运行，可以做到一次编译，处处运行，jvm就相当于一个翻译。

Java是一门纯面向对象语言，在Java中，处处皆对象。

Java的安装

大家自己去官网下载，然后安装，如果是windows系统，要设置环境变量，设置path、JAVA_HOME、classpath。不会的话请Google。

变量的命名规则

变量名就相当于一个篮子，来存放变量，它不能以数字开头，可以由数字、下划线、字母，$组成，Java中的变量名一般使用驼峰式命名。
