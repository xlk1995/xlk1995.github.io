变量与常量

常量

常量就相当于数学中常数。在Java中，声明常量要用final关键字，并且常量名哟啊大写，可以用public static final声明一个类常量，它在任何情况下都可以使用。分为以下6种：

  • 字符串常量
  • 整型常量
  • 浮点型常量
  • 字符常量
  • bool型常量 true和false
  • 空常量 null 不能直接打印

变量

变量就是可以改变的量，它的名字就是标识符，驼峰。变量需要先声明，再赋值，且必须赋值后才能使用。

变量需要注意的地方

  1. 如果创建多个变量，变量名不可重复
  2. 对于float\double\long 不要省略它们的后缀
  3. 使用基本数据类型，不能超过它们所表示的范围
  4. 没有进行赋值的变量不能直接使用
  5. 变量使用不能超过作用域

数据类型

java是一种强类型语言，这就意味着必须为每一个变量声明类型。

基本数据类型

基本数量类型有4类8种

整型

就是整数，默认为int。二进制以0b开头，八进制以0开头（不推荐使用，容易混淆），十六进制以0X开头。他们的取值范围就是-2^(占多少位-1)~2^(占多少位-1-1)，如byte的取值范围为-128~127。

  • byte 1字节 (byte)0 (default)
  • short 2字节 (short)0 (default)
  • int 4字节 0(default)
  • long 8字节 0L(default) 长整型后面需要L/l进行标识，推荐L

浮点型

浮点型就是小数，默认为double.它按精度可分为float和double两种，是近似值。

  • float 4字节 有效位数为 6 ~ 7 位 用F标识
  • double 8字节 有效位数为 15 位 用D标识

字符型

  • char 2字节 0~65535

布尔型

  • boolean 1字节 true 和 false

引用数据类型

字符串、数组、对象、lamada、类等等，非基本数据类型就是引用数据类型。

注意事项

  1. 字符串不是是引用数据类型
  2. 浮点数可能是一个近似值而非精确值
  3. 数据范围和占的内存没关系，比如float比long表示的范围更广

Java的起源

Java刚开始的名字叫做Oak，据说是它的创始人看窗外的时候正好有一颗橡树而起名的，它目标设定在家用电器等小型系统的编程语言，来解决诸如电视机、电话、闹钟、烤面包机等家用电器的控制和通讯问题。由于这些智能化家电的市场需求没有预期的高，Sun放弃了该项计划。就在Oak几近失败之时，随着互联网的发展，Sun看到了Oak在计算机网络上的广阔应用前景，于是改造了Oak，以“Java”的名称正式发布。

Java的特性

Java的第一个最出名的特性就是可移植性/跨平台，它可以在各种操作系统里面运行。java源文件编译成.class文件，通过jvm（Java Virtual Machine）在不同操作系统运行，可以做到一次编译，处处运行，jvm就相当于一个翻译。

Java是一门纯面向对象语言，在Java中，处处皆对象。

Java的安装

大家自己去官网下载，然后安装，如果是windows系统，要设置环境变量，设置path、JAVA_HOME、classpath。不会的话请Google。

标识符的命名规则

标识符就相当于一个篮子，它其实就是名字，来存放数据，它不能以数字开头，可以由数字、下划线、字母，$组成，Java中的标识符一般使用驼峰式命名。

语句

语句就相当于执行一件事情，如a = 2;就是把2赋值给a。

注释

Java中有三种注释方法。

  • //单行注释
  • /*....*/多行注释
  • /** ... */ 文档注释，可以通过javadoc命令生成文档。

Java的数据类型

Java的数据类型分为基本数据类型和引用数据类型，基本数据类型用来赋值，引用数据类型的变量名其实是变量在内存中存的地址。基本数据类型有八种，其他都是引用数据类型。

整数型有四种 byte short int long，位数分别为 8/16/32/64 ，默认值都是0。

小数型有两种 float doule，位数分别为 32/64，默认值为 0.0f和0.0d。它们能取得的最大最小值可以通过包装类的类名.MAX_VALUE/类名.MIN_VALUE获得。如果取值超出，数据便会溢出，+1便变为最小值。

boolean 有两种取值，分别为true和fasle，默认值为false。

char 是单个字符，默认值为''。

基本数据类型的转换

数据类型的转换默认是取值范围从小的转为大的，如果要把int转为long，则 int a = 9999999999; long b = (int)a。

流程控制

Java中的流程控制分为，顺序结构，分支结构，循环结构，控制循环结构。

顺序结构

顾名思义，顺序结构就是代码按顺序一行一行地执行。

分支结构

分支结构可以用if和switch两种方法来实现。

循环结构

循环结构用while，do while，和for循环来实现。

控制循环

有break和continue，break是跳出整个循环，continue是跳过当前循环。
